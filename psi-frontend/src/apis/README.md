# 前后端交互接口

该目录下存放前后端交互接口相关文件，包括交互函数定义以及界面数据模型定义，具体书写规则可以参考当前文件夹下面已经编写的代码

## 目录结构说明

目录与文件说明，一般按照模块名称划分目录，目录名称全部是小写字母，多个单词用-分割，所有ts文件也是全部小写字母命名

- `common`：公用数据相关接口、数据模型定义
- `login`：登录相关接口、数据模型定义
- `sample`：测试自定义组件时使用的相关接口、数据模型定义
- `????`：`????模块`相关接口、数据模型定义
- `http.ts`：二次封装`axios`相关代码定义，如无特殊情况一般不需要修改此文件
- `type.ts`：通用数据模型声明在这个文件里面，一般是一些父类模型和重复出现的模型，都声明到此处避免重复声明
