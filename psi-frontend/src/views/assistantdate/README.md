📋 项目概述
这是一个功能完善的企业管理系统，包含辅助资料管理、系统参数配置、业务操作等多个模块，为企业提供一体化的管理解决方案。

🚀 技术栈
前端框架: Vue 3 + TypeScript
UI 组件库: Element Plus
构建工具: Vite
状态管理: Vue 3 Composition API
样式预处理: Scss
图标库: Element Plus Icons


🎯 功能模块
1. 辅助属性管理(AssistantAttribute.vue)
功能描述: 管理商品的辅助属性（如颜色、尺寸等）
核心功能:
属性列表展示与搜索
属性的增删改查
属性内容管理（支持多值）
属性排序配置

2. 条码管理(BarcodeManage.vue)
功能描述: 管理商品条码和二维码信息
核心功能:
条码信息的增删改查
批量导入导出
条码预览功能
支持二维码和条形码类型

3. 常用功能配置(CommonFunction.vue)
功能描述: 配置用户常用的系统功能
核心功能:
功能模块分类展示
多选配置常用功能
配置持久化保存

4. 收支类别管理(IncomeCategory.vue)
功能描述: 管理企业的收入和支出类别
核心功能:
收支类别分类管理
类别排序配置
支持收入和支出两种类型

5. 商品类别管理(ProductCategory.vue)
功能描述: 管理商品分类体系
核心功能:
树形结构类别管理
类别层级关系维护
类别排序和备注信息
数据本地存储

🛠️ 核心组件
表格组件 (MyTable)
支持分页、排序、选择
自定义操作列
树形表格展示
斑马纹和边框样式

表单对话框 (MyFormDialog)
动态表单配置
表单验证规则
多种表单控件支持
响应式布局

报表表格 (ReportButtonTable)
内置操作按钮
批量操作支持
分页功能

🚀 快速开始
环境要求
Node.js 16+
pnpm 或 yarn
安装依赖
bash
pnpm install
开发环境运行
bash
pnpm dev
生产环境构建
bash
pnpm run build

📖 使用说明
1. 辅助属性管理
点击"新增"按钮创建新属性
在属性详情中添加属性内容值
使用搜索功能快速定位属性
支持属性的编辑和删除

2. 条码管理
支持单个条码的添加和批量导入
下载模板文件进行批量操作
查看条码详情和预览效果
导出数据进行分析

3. 功能配置
勾选需要显示的常用功能
点击"保存配置"应用更改
配置信息自动保存

4. 类别管理
使用树形结构组织类别
拖拽调整类别顺序
设置类别排序值
添加备注信息
