采购管理系统组件说明文档
项目概述
这是一个基于Vue 3 + TypeScript + Element Plus的采购管理系统，包含五个核心采购报表组件，提供完整的采购业务数据展示和分析功能。

组件列表

1. 采购订单跟踪表 (PurchaseOrder.vue)
   功能描述：
   树形结构展示采购订单及其明细项
   支持父子级数据汇总显示
   提供完整的搜索、筛选和排序功能

主要特性：
父子级数据展示（父级显示汇总信息，子级显示详细记录）
支持按供应商、商品、单据编号等多条件搜索
仓库筛选和排序功能
数据导出功能
分页显示

数据列：
所属组织、供应商、单据时间、单据编号
商品名称、辅助属性、仓库、单位
数量、金额、入库状态、未入库数量/金额
到货日期、备注信息

2. 采购明细表 (PurchaseDetail.vue)
   功能描述：
   平铺式展示采购明细数据
   区分采购单和采购退货单
   提供金额统计和分类汇总

主要特性：
单据类型区分显示
采购与退货金额分别统计
支持商品、供应商、单据类型等多维度筛选
金额颜色区分（正数为绿色，负数为红色）

数据列：
单位类型、所属组织、供应商
单据时间、单据编号、商品名称
辅助属性、仓库、单位、单价
数量、折扣额、金额、税额、价税合计
备注信息

3. 采购汇总表 (PurchaseSummary.vue)
   功能描述：
   按商品维度汇总采购数据
   对比采购和退货情况
   提供多维度筛选功能

主要特性：
按商品、供应商、用户、人员等不同维度筛选
采购与退货数据并列显示
汇总金额自动计算
弹窗式搜索界面

数据列：
关联人员、用户、供应商（根据筛选类型显示）
商品名称、辅助属性、仓库、单位
采购单（单价、数量、金额）
采购退货单（单价、数量、金额）
汇总（数量、金额）

4. 采购付款表 (PurchasePayment.vue)
   功能描述：
   跟踪采购付款进度和核销状态
   显示应付款余额和付款率
   提供付款进度可视化

主要特性：
付款进度条显示
核销状态跟踪
应付款余额统计
集成GoodSearch搜索组件

数据列：
单据类型、所属组织、供应商
单据时间、单据编号
单据金额、实际金额、单据付款
应付款余额、付款率、核销状态
备注信息

5. 采购排行表 (PurchaseList.vue)
   功能描述：
   按采购数量和价值进行商品排行
   提供成本分析和统计
   简洁的搜索和筛选功能

主要特性：
商品采购数量排序
成本与价值分析
简化的搜索界面
数据导出功能

数据列：
商品名称、辅助属性、商品编号
规格型号、单位、数量
折扣额、税额、价税合计
成本、总成本

通用功能
搜索组件 (GoodSearch)
所有组件都集成了统一的搜索组件，支持：
商品名称搜索
单据编号搜索
供应商筛选
日期范围选择
自定义字段扩展

数据操作
导出功能：支持CSV格式数据导出
刷新功能：重置搜索条件并刷新数据
分页功能：支持多种分页大小选择

界面特性
响应式设计，支持移动端显示
统一的表格样式和操作栏布局
金额格式化显示
状态颜色区分
加载状态提示

技术栈
前端框架：Vue 3 + Composition API
开发语言：TypeScript
UI组件库：Element Plus
样式：Scoped CSS
构建工具：Vite (推断)

数据格式
所有组件使用统一的数据结构：

typescript
interface BasePurchaseData {
id: number;
orgName: string;
supplierName: string;
productName: string;
// ... 其他通用字段
}

使用说明
安装依赖：确保已安装Vue 3、Element Plus等必要依赖
导入组件：在需要的地方导入相应组件
配置数据：根据实际业务需求配置数据源
自定义搜索：通过GoodSearch配置项自定义搜索条件
